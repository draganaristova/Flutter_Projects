# Projects for Mobile Information Systems

This repository contains projects developed for the course **Mobile Information Systems**.

## Projects

### 1. homework_exam_schedule
An exam schedule app.  
- List of exams with date, time, and rooms  
- Detailed view for each exam  
- Chronological sorting with different colors for past and upcoming exams  

### 2. displaying_recipes
A recipe app using **TheMealDB API**.  
- List of categories and meals by category  
- Detailed recipe view with ingredients, instructions, and YouTube link  
- Search functionality for categories and meals  
- Random recipe of the day  

### 3. displaying_recipes_with_firebase
An upgraded version of the previous recipe app with Firebase integration.  
- Add favorite recipes  
- Local or push notifications for a daily random recipe  
- Organized code with models, screens, widgets, and services
